## docker-compose

如果您还未安装 docker 可以查看我们：[安装 docker 文档](../../FQA/07.安装docker.md)

- 无需安装任何环境,自动编译构建

> 需要注意修改 `.env` 文件中的 token 值

```shell
git clone https://gitee.com/dromara/Jpom.git
cd Jpom
docker-compose up -d
```
