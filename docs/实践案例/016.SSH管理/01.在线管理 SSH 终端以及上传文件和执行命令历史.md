## 完美使用

1. 后续普通小文件可以直接使用 Jpom 系统上传、下载
2. 文本文件还可以在线编辑
3. Jpom 中还可以配置非管理员禁止执行某些命令,如：`rm`
4. Jpom 中还可以使用 SSH 命令管理来定时执行某些脚本
